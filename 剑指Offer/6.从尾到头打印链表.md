# 6. 从尾到头打印链表

## 题目描述

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

## 示例

> 输入：head = [1,3,2]
> 输出：[2,3,1]

## 限制

> 0 <= 链表长度 <= 10000

## 代码

```java
public static int[] reversePrint(ListNode head) {
    ListNode now = head;
    int i = 0;
  	// 先计数
    while (now != null) {
        i++;
        now = now.next;
    }
    int[] ans = new int[i];
    now = head;
  	// 倒转存入数组
    while (now != null) {
        i--;
        ans[i] = now.val;
        now = now.next;
    }
    return ans;
}
```
